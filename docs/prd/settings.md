# PRD: 设置 settings（`/dashboard/settings`）

> 见 [README](./README.md)。本轮只定产品功能范围，不含实现。新功能改本文件。

## 目的
host 维护自己的身份与联系方式。

## 当前功能（已建）
显示名 · 唯一用户名（实时校验，启用 /u/ 主办主页）· 微信号(R4)。

## MVP 需要的功能（精简 + 对齐双盲）
- **昵称合并**：原"用户名 + 显示名" → **单一"昵称"**（非唯一显示名，不再需要唯一 handle）。
- **设置 = 昵称 + 微信 + 通用联系方式**。host 的微信/通用联系方式同样**成局后向访客解锁**（与访客侧对称、双盲一致）。
- **砍掉 /u/ 主办主页**（§5 入口是局不是人；代码保留、MVP 不出口；活动页"由 X 主办"变纯文本不链 /u/）。
- **头像**：暂不加（留空、保留未来扩展位）。

## 缓做 / 不做
host 简介(bio)（V2）· 邮箱后缀认证（V2；未来可先给 host 由登录邮箱域推导）· 默认城市/默认局卡偏好 · 通知偏好。
