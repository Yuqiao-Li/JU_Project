# PRD: 仪表盘 dashboard（`/dashboard`）

> 见 [README](./README.md)。本轮只定产品功能范围，不含实现。新功能改本文件。

## 目的
host 的主页：管理自己的局、快速再开局；也聚合"我参加的局"。

## 当前功能（已建）
"我主办 + 我参加"列表（即将/过去，角色徽章，人数计数）· 新建局 CTA · 空态引导。

## MVP 需要的功能（= 当前保留 + 新增）
- **新增·顶部局卡**：放"当前/最新局"的**局卡**（模块化组件，可复用到管理台）；其余历史局走现有紧凑列表。
- **新增·一键复用入口**：历史局"再开一局"→克隆预填、时间顺延→改时间即发。
- 保留"我主办 + 我参加"列表。

## 缓做 / 不做
历史票根墙（§4，V2）· "我参加的局"做成票根收藏（V2）。

## 依赖的成局契约
