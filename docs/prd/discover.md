# PRD: 发现 discover（`/discover`）

> 见 [README](./README.md)。本轮只定产品功能范围，不含实现。新功能改本文件。

## 目的
公开浏览站点级的局。MVP 保持简单（JU 主要靠链接/二维码分享，不做大广场）。

## 当前功能（已建）
公开活动网格（即将/已过去，封面/标题/主办/时间/城市卡片）· RPC 仅返回 public+published。

## MVP 需要的功能（= 当前保留 + 轻改）
- **紧凑局卡**列表（局卡组件的紧凑变体）。
- **静默隐藏过滤**：未成局/过期的局不在公开列表出现（§5/§6 无社死）。

## 缓做 / 不做（cluster A 缓做）
情绪胶囊 Vibe Filter · 双层黑话标签筛选 · 宏观城市 + 微观邻里过滤（§10）· 城市局单策展 · 把发现页升级为"懂你的"局卡流。

## 依赖的成局契约
静默隐藏、局分类（未来筛选）—— 见 [README 跨页契约](./README.md#跨页成局契约)。局卡见 [event-card](./event-card.md)。

## 品牌基调约束
待 PDF：发现页调性（克制、工具感、非广场）。

## 实现逻辑（Step-10A 设计）
- **紧凑局卡** = 复用局卡组件（紧凑变体）。
- ⚠️ **成局/未成局不能用 is_locked 判**（R4 自动锁让临近开始的局都 is_locked）；"成局" = 凑满(going≥capacity) OR host 手动锁(`locked_at` 非空)，自动锁不算。
- **静默隐藏** = 未成局（过期 + 设了 capacity + going<capacity + `locked_at` 空）→ `get_public_events` 加一条 WHERE 隐藏；host 仍 dashboard 可见可复用。开放局不受限。

## 状态
功能范围 + 实现逻辑已定（Step 10A）。紧凑局卡 + 静默隐藏过滤 未开始。
