# PRD: 建局 event-create（`/dashboard/events/new`）

> 见 [README](./README.md)。本轮只定产品功能范围，不含实现。新功能改本文件。

## 目的
主办单屏建局；最低摩擦产出一个可分享、有成局目标的局。

## 当前功能（已建）
标题/描述、封面/主题色/特效、时间(+TBD/日期投票入口)、地点(城市/地址/地图链接)、容量、+1 携伴、
RSVP 收集开关、Chip-in 收款链接、公开/私密、密码保护、主办微信(R4)、发布/草稿。

## MVP 需要的功能（= 当前保留 + 新增/改）
- **新增·成局目标"缺几人"**：局的目标人数；全员见"缺 X 人"，凑满为目标（喂养局卡进度、自动成局提示、静默隐藏）。
- **新增/改·选分类 + 选局卡**：建局时**选局分类** → 平台**自动生成多款局卡**供 host 选（**不传图**，审美兜底）。
  分类同时沉淀后台 category（为未来发现/推荐蓄水）。现"外观段(封面/主题/特效)"被"**选局卡**"取代/吸纳。
- **改·发布必填** = 标题 + 微信 + 时间(或显式 TBD) + 城市（草稿不强制）。
- 其余字段按现状保留（option A：结构不改，逐字段为实现期细节）。

## 缓做 / 不做
四核心+高级折叠重排 · 粘贴文字一键建局 · 黑话占位文案（绑 PDF）· 波普纯文字 fallback 卡 · 邻里/通勤圈字段（§10）· host 传图包装。
